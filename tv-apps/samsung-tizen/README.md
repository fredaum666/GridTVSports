# GridTV Sports - Samsung Tizen TV App

A WebView wrapper for Samsung Smart TVs (2015+).

## Prerequisites

- Tizen Studio: https://developer.tizen.org/development/tizen-studio/download
- Samsung Developer Account: https://developer.samsung.com/
- Samsung TV in Developer Mode

## Enable Developer Mode on Samsung TV

1. Go to Apps
2. Press 1, 2, 3, 4, 5 on remote
3. Set Developer Mode ON
4. Enter your PC's IP address
5. Reboot TV

## Project Setup

1. Open Tizen Studio
2. File > Import > Tizen Project
3. Select this folder

## Building & Running

1. Connect to TV: Tools > Device Manager > Remote Device Manager
2. Add your TV's IP
3. Right-click project > Run As > Tizen Web Application

## Publishing

Submit to Samsung Galaxy Store:
https://seller.samsungapps.com/tv
